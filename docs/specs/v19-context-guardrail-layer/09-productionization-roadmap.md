# 09 - Productionization Roadmap

## Phase 1

- SQLite memory store。
- deterministic ContextPack。
- context guardrails。
- supply chain MVP integration。

## Phase 2

- SearchRouter 自动写入 Evidence Memory。
- Graph Memory 与 Neo4j 同步。
- FinRisk workflow 接入。
- API context preview。

## Phase 3

- Episodic memory。
- user feedback。
- negative memory。
- frontend Context Drawer。

## Phase 4

- embedding reranker。
- vector index。
- per-source TTL。
- provider cost budget。
- source spam detection。
- memory poisoning quarantine。

## Production 指标

- context selection latency。
- selected / rejected memory count。
- stale memory rate。
- unsupported claim count。
- graph edge evidence coverage。
- memory poisoning quarantine count。
- human feedback correction rate。
